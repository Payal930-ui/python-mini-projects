# 🐍 Python Mini Projects Collection

## 📌 Overview

This repository contains a collection of beginner-friendly Python mini projects developed to strengthen core programming concepts and problem-solving skills. The projects cover topics such as conditional statements, loops, functions, randomization, user input handling, and basic data management.

These projects were built as part of my Python learning journey and demonstrate practical implementation of fundamental programming concepts.

---

## 📂 Projects Included

### 🏠 Rent Calculator

A simple application to calculate the rent share for each person based on total expenses and the number of roommates.

### 🔐 Password Generator

Generates secure and random passwords with customizable length and character combinations.

### 🎯 Number Guessing Game

An interactive game where users try to guess a randomly generated number with hints for guidance.

### ✊✋✌️ Rock Paper Scissors Game

A classic game where the user plays against the computer using random choices and game logic.

### 🐍💧🔫 Snake Water Gun Game

A fun Python implementation of the traditional Snake-Water-Gun game against the computer.

### 🎓 Student Grade System

Calculates grades based on marks and displays performance according to predefined grading criteria.

### ✅ Task Management System

A simple task manager that allows users to add, view, update, and remove tasks efficiently.
